# tinydoc
Minimal source code documentation generator used for [LEKTRA](https://github.com/dheerajshenoy/LEKTRA) [configuration page](https://dheerajshenoy.github.io/lektra) generation


# Keywords

- `@desc` - Description
- `@choice` - Options to choose from
- `@return` - Return value
- `@type` - Datatype
- `@section` - Beginning of section
- `@endsection` - End of section
- `@section_desc` - Description of section
- `@section_type` - Type of section
